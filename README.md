## 執行方式

1.請先確認有安裝 node.js 與 npm

2.將專案 clone 到本地

    git clone https://github.com/sigox/3S2A5.git

3.安裝 npm 套件

    npm install

4.啟動專案

    npm start
    
5.若欲暫停使用
